
sandbot-wptgen
--------------
Sandbot waypoint generator.

Based off bsp_tools by Jeffrey 'botman' Broome.


Usage
=====
sandbot-wptgen -wN mapname.bsp

A larger value of N will result in faster waypoint generation.


Tests
=====
powershell ./test.ps1
